## HW_API
### Create a simple that includes the  following APIs:
1. POST: /data: This api should return the data sent to it in JSON format in addition to your name, and it should return it in JSON format  too, for example If I sent {“message”:”hello”} I should get the{“message”:”hello”, “name”:”Ahmed”}

2. GET: /greeting/<name>: This api should return a greeting message with the name sent in the URL, for example if I sent a GET request to /greeting/Ali I should get {“message”:”Hello Ali”}
Note: You can either create the backend with Node JS with express or with pure PHP (No framework allowed for PHP)

3. write a README file explaining how to run and test your project
